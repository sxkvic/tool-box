# 随身工具箱 · 微信小程序

纯前端个人工具合集，**无需后端 / 无需服务器**。内置流量主广告位预留，累计 UV 达标后填入广告 ID 即可变现。

## 体验能力

- 深空 / 纯白主题切换（右上角，含过渡动画）
- 首页 **三分类导航** + **最近使用**、**收藏置顶**
- 每日次数分层：轻量工具免费，抽签/生成/试算计次
- 各工具页支持分享好友 / 朋友圈
- 纯本地 `wx.storage`，无后端依赖

## 快速开始

1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开开发者工具 → **导入项目** → 选择本目录
3. `AppID`：
   - 测试：使用测试号 / `touristappid`
   - 正式：换成你在[公众平台](https://mp.weixin.qq.com/)注册的小程序 AppID
4. 编译预览即可（无需配置服务器域名）

## 广告（流量主）接入

1. 小程序上线并获得足够真实访问（官方常见门槛为累计 UV ≥ 500，以后台为准）
2. 公众平台 → **推广 → 流量主** → 开通
3. 新建广告位后，编辑 `config/ads.js` 填入广告位 ID
4. 未填写 ID 时广告自动隐藏，工具照常可用

```js
dailyFreeQuota: 12,   // 计次工具每日免费次数
rewardExtraQuota: 5,  // 激励视频奖励次数
```
